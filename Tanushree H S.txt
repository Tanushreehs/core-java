1)Java is a high level ,objected-oriented programming language. It is platform independent because java compiles bytecode which is then executed by JVM.JVM is supported by multiple operating systems hence making it platform independent.

2)JDK architecture contains the following tools:
JDK : used for compilation(javac),or debugging.
JRE : provides runtime environment
JVM : executes bytecode ,helps in storage, security and makes java platform independent.

3)JVM(java virtual machine) is a virtual machine that executes java bytecode. It converts bytecode into machine code and helps in executing java code. It plays a crucial role in making java a platform independent language as it is supported in multiple OS.

4)Here are the steps of "Write Once ,Run Anywhere":
Write the code and save the code using the extension (.java)
Compile the code using a tool(javac filename.java)
Execute the code using (java filename)

5)Bytecode is the intermediate code in java generated just after compilation . It is then converted into machine code by JVM.

10)class Shopkeeper {
    public static void main(String[] args) {
       int[] sales = {100, 200, 150, 300, 250, 400, 350};

        int total = 0;
        int max = sales[0];
        int day = 1;
        for(int i = 0; i < sales.length; i++) {
            total += sales[i];

            if(sales[i] > max) {
                max = sales[i];
                day = i + 1;
            }
        }

        System.out.println("Total Sales: " + total);
        System.out.println("Highest Sale: " + max);
        System.out.println("Day: " + day);
    }
}
